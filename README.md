# WebDesign Black

В этом репозитории находится веб-страница в стиле "black", которая используется для демонстрации моих навыков в веб-дизайне.

![Скриншот веб-страницы](https://timqwees.github.io/WebDesign_Black/photo.png)

На странице представлены различные элементы, такие как заголовки, списки, таблицы и изображения. Я также использовал SASS для стилизации страницы.

Вы можете открыть страницу на своем компьютере, перейдя по [этой ссылке](https://timqwees.github.io/WebDesign_Black/).
